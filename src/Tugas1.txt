2. Sebutkan dan jelaskan 13 method-method string dalam Java
    = length() memberikan informasi akan panjangnya string.
      charAt(int index) mengambil huruf/karakter di posisi tersebut.
      equals(Object obj) membandingkan strings.
      toLowerCase() mengubah string menjadi huruf kecil
      toUpperCase() mengubah string menjadi huruf kapital
      trim() menghapus spasi di awal dan akhir string.
      replace(CharSequence target,CharSequence replacement) mengganti substring dengan yang lain.
      split(String regex) memecah string berdasarkan regex ("Hello World" -> "Hello" "World")
      indexOf(String str) mengembalikan posisi pertama substring
      isEmpty() mengecek string apabila kosong
      substring(int beginIndex, int endIndex) mengambil potongan dari string.
      contains(CharSequence seq) mengecek apabila string ada substring.
      equalsIgnoreCase(String anotherString) membandingkan string tanpa memperhatikan huruf besar/kecil.
